# Downloads für Veranstalter

Dieser Ordner enthält alle wichtigen Dokumente für Veranstalter und Techniker.

## Verfügbare Downloads

### Technical Rider
**Datei:** `technical-rider.pdf`

Enthält alle technischen Anforderungen:
- Bühnenmaße und Aufbau
- Stromanschlüsse
- Monitoring-Anforderungen
- Soundcheck-Zeiten
- Besondere technische Anforderungen

### Stageplot
**Datei:** `stageplot.pdf`

Detaillierter Bühnenplan mit:
- Positionierung aller Musiker
- Instrumenten-Setup
- Mikrofonierung
- Kanalliste
- Kabelwege

### Pressekit
**Datei:** `pressekit.zip`

Komplettes Pressepaket mit:
- Bandbiografie
- Hochauflösende Bandfotos
- Logo in verschiedenen Formaten (PNG, SVG, JPG)
- Pressetexte (kurz, mittel, lang)
- Technische Daten
